3个重要组成部分

- custom elements  
  ie 不支持，chrome 63 以后的版本都支持
- shadow DOM trees  
  ie 不支持，chrome 63 以后的版本都支持
- template & slot  
  Edge 15 以后才完全支持，其余浏览器基本都支持
